# Rsg-brand-extension Extension For Quarto

This extension adds a default RSG theme to quarto projects. The brand file uses
RSG's colors, logos, and typography.

## Installing

```bash
quarto add RSGInc/rsg-brand-extension
```

This will install the extension under the `_extensions` subdirectory.
If you're using version control, you will want to check in this directory.

## Using

This extension installs a [brand.yml](https://posit-dev.github.io/brand-yml/) configuration for RSGInc.
